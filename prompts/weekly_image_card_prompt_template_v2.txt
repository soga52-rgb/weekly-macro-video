STYLE_BASE = """
Create a 16:9 NotebookLM-style whiteboard explainer image for a weekly macro video.

Visual style:
- off-white / white background
- subtle light gray grid-paper texture
- bold black hand-drawn line art
- orange accent marks, arrows, labels, pins, circles
- clean macro-finance explainer diagram
- looks like a knowledge video card, not a corporate slide
- use icons, causal arrows, simple doodles, flow paths, question marks, magnifying glass, warning tags
- keep the composition airy and video-friendly

Important:
This image will be placed inside a video.
Visible text must be very short, large, and readable.
Do not create a dense report.
Do not create tables.
Do not use long paragraphs.
Do not add unnecessary small text.
"""

TEXT_RULES = """
Visible text rules:
- Use Traditional Chinese only.
- Maximum one main headline.
- Use only short labels.
- Each visible label should be under 12 Chinese characters when possible.
- No paragraph-style text.
- Do not invent new numbers.
- Do not change asset directions.
- If text is hard to fit, prioritize diagrams and icons over long words.
- The image should communicate the logic visually, not by reading a lot of text.
"""
